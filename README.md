# CM-SimulatorGUI

![alt text](CMSimulator.png)

Cache Memory Simulator

## Specs

- Words Size: 4/8 bytes

- Block Size: 32/64 bytes

- Set Size: 1/2/4/8 lines

- Replacement Policy: FIFO or LRU
